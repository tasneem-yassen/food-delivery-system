# Food Delivery System - Java OOP Project

## About
A Java object-oriented food delivery system that manages restaurants, customers, orders, and riders.

## Features
- Create and manage orders
- Assign available riders to orders
- Track rider availability
- Use inheritance and method overriding
- Use encapsulation with getters and setters
- Model real-world relationships between classes

## OOP Concepts Used
- Classes and objects
- Inheritance
- Polymorphism
- Method overriding
- Encapsulation
- Class relationships / UML

## Technologies
- Java
- Eclipse IDE
- Git & GitHub

This project is still in progress and will be expanded in future homework assignments.
