# Food Delivery System - Java OOP Project

## About
A Java-based object-oriented food delivery and order management system developed as part of a university OOP course.

The system simulates a real-world food delivery platform that manages:
- Customers
- Restaurants
- Riders
- Orders
- Restaurant managers
- System administrators

The project focuses heavily on object-oriented programming principles and system design.

---

## Main Features

### Customer Features
- Create food orders
- View order history
- Update personal information
- View restaurant details

### Rider Features
- View assigned deliveries
- Update delivery status
- Manage active orders

### Restaurant Manager Features
- Add restaurants
- Add customers
- Create orders
- Assign riders to deliveries

### Admin Features
- Manage the entire system
- Add riders, restaurants, and managers
- Assign managers to restaurants
- Manage delivery assignments

## Restaurant Types
The system supports multiple restaurant types using polymorphism:

- Regular Restaurant
- Fast Food Restaurant
- Premium Restaurant

Each restaurant type calculates delivery pricing differently.
